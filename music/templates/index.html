IndyKite Music

Use Knowledge-Based Access Control (KBAC) and ContX IQ (CIQ) to retrieve or upsert music data according to context. CIQ provides fine-grained, context-aware data access across the music graph: people, artists, tracks, albums, venues, playlists, subscriptions and the relationships between them.

How ContX IQ Works
1. Policy

Defines what data can be accessed and who can access it. Uses Cypher patterns to match graph structures and filters to bind parameters. Specifies allowed reads, upserts, and deletes.

2. Knowledge Query

Defines what data to return from the matched graph. Specifies which node properties and relationships to include in results.

3. Execute

Runs the query with input parameters. The policy validates access, the query retrieves data, and results are returned. Uses the application's credentials for authorization.
