An MCP Server configuration tells IndyKite which App Agent and Token Introspect to use when authenticating inbound MCP traffic for this project. Create one after the App Agent and Token Introspect already exist.

The form is pre-populated using APP_AGENT_ID and TOKEN_INTROSPECT_ID from .env when those steps were completed in this app. The created MCP Server's id is saved as MCP_SERVER_ID.
