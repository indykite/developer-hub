Create External Data Resolver

An External Data Resolver fetches data from a remote system at query time. Pair it with a node property using external_value instead of value so the property is resolved on demand.

This form is pre-populated with a weather resolver that calls open-meteo for CanBank's London headquarters and returns the current temperature. The same public endpoint is used by the iag-demo weather agent.

Fill in the fields and click "Create" to submit a POST /configs/v1/external-data-resolvers request.

Used as the external_value on a node property.
Map of header name to a list of values, e.g. {"Authorization": ["Bearer token"]}.
